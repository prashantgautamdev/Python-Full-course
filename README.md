## 📁 Repository Structure

Each chapter file follows the same structure:

1. **Notes** — a short Hinglish explanation of the topic with runnable example code (kept as a docstring at the top of the file).
2. **Practice Questions** — Basic → Intermediate → Advanced, no answers.
3. **Project Ideas** — a few small applied projects to build using that topic.

## 🚀 How to Use

1. Clone or download this repository:
   ```bash
   git clone https://github.com/<your-username>/<your-repo-name>.git
   cd <your-repo-name>
   ```
2. Open any chapter file and pick a question.
3. Write your solution directly below the question (or in a separate file) and run it:
   ```bash
   python chapter1.py
   ```
4. Track your progress by checking off questions as you solve them.
